## Example Project Name: Personal Career Journey Profile Website

Business Domain: Personal Branding / Professional Portfolio / Career Profile

Technology Stack: HTML, CSS, JavaScript

## Scope

- Create a static responsive personal profile website.
- Present career journey from past jobs to present.
- Use a dark blue visual theme.
- Use HTML, CSS, and JavaScript only.
- Reference structure and content from the provided profile website.
- Include professional sections for introduction, about, timeline, job experience, domain exposure, credentials, education, and contact.
- Support desktop and mobile screen sizes.

## Mandays

- Requirement Gathering: 0.5 day
- UX/UI Design: 0.5 day
- SA Design: 0.5 day
- Development: 1 to 2 days
- Manual QA: 0.5 day
- Automation QA: 0.5 day
- Final Review: 0.5 day

Estimated total: 4 to 5 working days

## Business Goals

- Present a clear professional identity as a Senior Software Tester and System Analyst.
- Show a career journey from QA foundation to broader system analysis and delivery support.
- Highlight domain exposure across telecom, insurance, and e-commerce.
- Make career timeline and job experience easy to scan.
- Provide a polished personal brand with a calm dark blue theme.
- Provide clear contact access for professional opportunities.
- Support desktop and mobile visitors.

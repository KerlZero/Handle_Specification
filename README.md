## Example Project Name: Personal Career Journey Profile Website

Business Domain: Personal Branding / Professional Portfolio / Career Profile

Technology Stack: HTML, CSS, JavaScript

## Scope

- Create a static responsive personal profile website.
- Present career journey from past jobs to present.
- Use a dark blue visual theme.
- Use HTML, CSS, and JavaScript only.
- Reference structure and content from the provided profile website.
- Include professional sections for introduction, about, timeline, job experience, domain exposure, credentials, education, and contact.
- Support desktop and mobile screen sizes.
